# TI-83 Congruence Calculator

A lightweight Python program for the **TI-83 Premium CE** calculator, designed to quickly compute modular arithmetic expressions and congruences.

This tool is useful for mathematics exercises involving congruences, modular arithmetic, powers modulo `n`, and quick verification of results directly on a calculator.

## Features

- Compute expressions modulo `n`
- Supports additions and subtractions
- Supports powers using `^` or `**`
- Automatically detects modulo notation in the input
- Optimized modular exponentiation for faster calculations
- Designed for TI-83 Premium CE Python
